# AI Interface Component Library

Welcome to the AI Interface component library! This Storybook contains reusable components built with **React**, **TypeScript**, and **Tailwind CSS**.

This component library was created as part of a Frontend & UI/UX Designer Assessment. It demonstrates:

- Modern UI patterns from leading AI platforms (ChatGPT, Claude, HuggingFace)
- Accessible, keyboard-navigable components
- Consistent design system using Tailwind CSS
- TypeScript for type safety
- Interactive component documentation

## ðŸ§© Components

### Button
Versatile button component with multiple variants (primary, secondary, outline, ghost) and sizes (sm, md, lg). Supports icons and disabled states.

### Slider
Range input component for parameter controls like temperature and max tokens. Features custom styling and accessibility support.

### ChatBubble
Message bubble component for chat interfaces. Displays user and assistant messages with distinct styling, avatars, and copy functionality.

### Modal
Dialog component with backdrop and multiple size variants. Includes header with close button and scrollable content area.

## ðŸŽ¨ Design System

### Colors
- **Primary**: Blue (bg-blue-500, bg-blue-600)
- **Secondary**: Gray (bg-gray-200, bg-gray-700)
- **Success**: Green (text-green-600)
- **Dark Mode**: All components support dark mode using Tailwind's `dark:` variants

### Typography
- **Headings**: text-lg, text-2xl with font-semibold or font-bold
- **Body**: text-sm, text-base with regular weight
- **Labels**: text-xs with uppercase and font-semibold

### Spacing
Consistent spacing scale using Tailwind: 2px, 4px, 8px, 12px, 16px, 24px, 32px

### Accessibility
- All interactive elements have proper ARIA labels
- Keyboard navigation support
- Focus states with visible ring indicators
- Screen reader friendly semantic HTML
